💡 Sobre o Projeto

O Sistema Seguro é um projeto web focado em autenticação de usuários, com destaque para um diferencial importante:

👉 Análise de força da senha em tempo real

Durante o cadastro ou login, o sistema identifica automaticamente o nível de segurança da senha digitada, classificando como:

🔴 Fraca
🟡 Média
🟢 Forte

Isso ajuda o usuário a criar credenciais mais seguras e melhora a proteção geral da aplicação.
